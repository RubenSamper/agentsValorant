# Actividad 4

Proyecto React con Vite listo para producción y despliegue en GitHub Pages.

## Desarrollo

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

El resultado queda en `dist/`.

## Publicación en GitHub Pages

1. Sube el repositorio a GitHub.
2. Ejecuta `npm run build`.
3. Publica la carpeta `dist/` en GitHub Pages con el flujo que prefieras.

La configuración de Vite usa rutas relativas para que el sitio funcione correctamente en GitHub Pages.
